Introduction

Sndsvr is an API for playing the audio from Youtube videos on speakers remotely. If the audio has never been played before, it downloads the audio first and then plays it. The audio is cached for future use.

    Data Structure

    Key

    Required: yes
    Type: string
    Accepted Values: any string

    The password for accessing the sound server. This attribute must always be at the bottom layer of the JSON object.

    ID

    Required: yes
    Type: string
    Accepted Values: youtube video ID

    The Youtube video ID to be played. The video ID can be found by looking at the URL. A standard Youtube URL is in the format "http://www.youtube.com/watch?v=xxxxxxxxxx" where the x's are the video ID.

    Volume

    Required: no
    Type: number int
    Accepted Values: [0, 100]

    The volume to play the audio at. The default is 50.

    JSON Examples

    A command that plays the "Wow! Montage Parody" audio at volume 60 looks like this:

    {
    "Key": "yourpassword",
    "ID": "Xm_dS-wEFvs",
    "Volume": 60
    }
