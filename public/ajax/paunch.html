Paunch

Paunch is a game library written in Go. I'm writing it primarily for my own use later on, but it's written with the intent that it'd be usable as a general 2D game engine. The Go code can be found on my Github here!

Paunch's power comes from it's simplicity. I've tried to put power in the hands of the user when needed, and automate the rest. The following code opens a new window.

import "github.com/velovix/paunch"

func main() {
paunch.SetWindowSize(960, 540)
paunch.SetWindowTitle("Trinosaurs")

paunch.Start()

for !paunch.ShouldClose() {
paunch.UpdateEvents()
}
}

The Paunch User Guide goes into much more detail about how you can use Paunch for your project. If you're interested, please check it out!

There also exists an unfinished C++ implementation found here, but it mostly exists for experimentation purposes. It may be buggy or unstable, so I would not recommend using it as anything but a base for your own development.
