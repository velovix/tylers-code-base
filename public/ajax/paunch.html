Paunch

Paunch is a game library written in Go and C++. I'm writing it primarily for my own use later on, but it's written with the intent that it'd be usable as a general 2D game engine. The Go code can be found on my Github here and the C++ code is here!

In the future, the C++ implementation of Paunch will be primarily supported, but I may update the Go implementation sometime in the future.

Paunch's power comes from it's simplicity. I've tried to put power in the hands of the user when needed, and automate the rest. The following code opens a new window in C++.

#include <paunch.h>

int main() {
paunch::Window window(640, 480, 640, 480, false, "Test Window");

window.Open();

while(!window.ShouldClose()) {
window.UpdateEvent();
}
}
